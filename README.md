# Claude Code 懶人包

> 每支教學影片都會附一個 MD 檔（懶人包），丟給 Claude Code 桌面版就能自動完成。

---

## 使用方式

1. 看影片了解原理
2. 下載對應的懶人包（MD 檔）
3. 打開 Claude Code 桌面版（Pro 以上），把檔案丟給它
4. AI 自動執行，遇到需要手動操作的地方會暫停指示你

---

## 先備條件

在使用任何懶人包之前，請確認以下準備工作已完成：

- [ ] Claude 帳號已註冊（Pro 方案以上）
- [ ] Claude Code 桌面版已安裝
- [ ] 已看過 EP01（了解 Chat / CoWork / Code 三個分頁）
- [ ] 已看過 EP02（了解什麼是 Skill）
